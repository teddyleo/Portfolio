Maze Generator

This implementation uses a depth-first search algorithm and a recursive backtracker to generate the maze. It then does a depth-first search on the generated maze to complete it.

Game of Life

This is an implementation of Conway's Game of Life, where a two-dimensuional grid consists of cells that can be either alive or dead. Click the canvas to generate a random set of alive/dead cells.

Tree Generation

This sketch uses a L-System to generate a collection of production rules in the form of a string, which is used to create the tree-like structure.
